# Security

Please do not open a public issue containing API keys, access tokens, private media, local filesystem paths that reveal sensitive information, or proprietary project data.

CutPilot executes local media tools and may connect to providers explicitly configured by the user. Review external endpoints and generated-media costs before enabling them. External Remotion dependency installation requires explicit approval, and lifecycle scripts are disabled by default.

Until a private security contact is published, report security issues through a private GitHub security advisory for this repository.
