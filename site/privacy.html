本地处理

CutPilot 是本地优先的视频剪辑引擎。项目文件、素材分析、时间线修改、人工审阅界面与本地渲染默认在你的设备上完成。

远程服务

只有当你主动选择远程转录、视频生成、图像生成、音乐、音效或其他第三方服务时，完成请求所需的输入才会发送给对应服务。第三方服务受其自身隐私政策和条款约束。

AI 主机

Codex、Claude 或其他 MCP 主机可能接收你在对话中提供的文字、文件路径、工具结果或你明确允许读取的内容。请根据所用主机的设置管理数据与权限。

网站

当前官网不设置广告 Cookie，也不要求账户登录。Cloudflare 可能为安全、网络交付与运行诊断处理常规请求信息。

密钥

API 密钥应保存在本机环境变量或受支持的秘密管理系统中。请勿把密钥提交到 GitHub 或写入 CutPilot 项目文件。

联系与变更

如有隐私问题，请通过 CutPilot GitHub 仓库提交问题。本说明会随着远程功能和网站能力变化而更新。
